CodeBhasha is an innovative Android app designed to make computer coding accessible in native Indian languages, starting with Malayalam. Traditionally, learning to code has required proficiency in English, which poses a barrier for many students, especially in rural or non-English-speaking regions. Recognizing that students learn better when taught in their mother tongue, this app aims to bridge the gap between language and technology education.

The core of this idea lies in the belief that language should not be a barrier to learning technology. CodeBhasha removes the dependency on English by allowing users to write and understand code in their own language. The first implementation, called ezr Malayalam, is a simple programming language built for the Malayalam-speaking community. It allows children to grasp key programming concepts using familiar words and logic, making the learning experience more intuitive and enjoyable.

What makes this app even more powerful is its accessibility. It runs on Android smartphones, eliminating the need for expensive computers or laptops. This opens up opportunities for students in economically weaker or remote areas to learn coding without major infrastructure investments. The programming language itself is built using Dot Net C#, and the app was developed using Unity, making it flexible and easy to adapt for other languages like Hindi, Tamil, or even Sanskrit by changing the syntax.
